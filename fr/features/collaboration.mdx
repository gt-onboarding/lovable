Lovable vous permet de créer des sites web ensemble. Invitez votre designer, développeur, agence, ou toute autre personne. Tout le monde voit les modifications au fur et à mesure.

Il existe deux façons de collaborer. Vous pouvez inviter des collaborateurs sur un projet spécifique ou dans l’ensemble de votre espace de travail.

<div id="collaboration-in-a-specific-project">
  ## Collaboration sur un projet spécifique
</div>

Vous pouvez inviter des utilisateurs à collaborer sur un projet spécifique en cliquant sur « Invite » dans l’éditeur de projet.
Ajoutez leur adresse e-mail : ils recevront ensuite une invitation par e-mail.

<Note>
  Les collaborateurs utiliseront les crédits de l’espace de travail du propriétaire du projet.
</Note>

<div id="collaboration-in-a-workspace">
  ## Collaboration dans un espace de travail
</div>

<div id="what-is-a-workspace">
  ### Qu'est-ce qu'un Workspace ?
</div>

Un Workspace est votre espace partagé pour créer et collaborer sur des projets. Vous pouvez inviter vos coéquipiers dans votre Workspace, et ils auront accès à tous les projets qui s'y trouvent.
Ils peuvent voir, modifier et publier les projets du Workspace.

<Warning>
  Pour un contrôle fin des permissions et l'attribution de rôles dans votre Workspace, un abonnement payant est nécessaire.
</Warning>

Quand vous vous abonnez à une offre, elle est associée au Workspace souhaité, et tous les projets du Workspace utilisent alors ces crédits, peu importe qui envoie les messages.

Vous pouvez personnaliser votre Workspace dans les paramètres du Workspace en :

- le renommant
- changeant l'avatar
- ajoutant une description

<div id="how-do-i-add-and-manage-collaborators-in-a-workspace">
  ### Comment ajouter et gérer des collaborateurs dans un espace de travail ?
</div>

Vous pouvez ajouter et gérer des collaborateurs depuis l’onglet **People** dans les paramètres de l’espace de travail. Vous pouvez y accéder via **Settings** \> **People**.

Ici, vous pouvez :

- Ajouter de nouveaux collaborateurs en saisissant leur e‑mail. Ils recevront une invitation par e‑mail qu’ils pourront accepter.
- Gérer les collaborateurs existants en modifiant leurs rôles ou en les supprimant.
- Voir l’utilisation mensuelle des crédits pour chaque collaborateur.

<div id="faq">
  ## FAQ
</div>

<AccordionGroup>
  <Accordion title="Combien de membres un espace de travail peut-il avoir&nbsp;?">
    Les espaces de travail prennent en charge un nombre illimité de membres sur toutes les formules. Pour les espaces de travail avec une formule Enterprise, vous pouvez définir des limites personnalisées pour le nombre de membres.
  </Accordion>
  <Accordion title="Puis-je avoir plusieurs espaces de travail&nbsp;?">
    Oui, mais vous aurez besoin d’abonnements séparés pour chacun.
  </Accordion>
  <Accordion title="Puis-je basculer entre plusieurs espaces de travail&nbsp;?">
    Oui&nbsp;! Accédez au tableau de bord principal. De là, cliquez sur le nom de votre espace de travail dans le coin supérieur droit. Dans le menu déroulant, vous verrez les autres espaces de travail auxquels vous avez accès. Pour changer, il vous suffit de cliquer sur l’espace de travail souhaité.
  </Accordion>
  <Accordion title="Est-il possible de transférer des projets entre espaces de travail&nbsp;?">
    Oui&nbsp;! Accédez aux paramètres du projet, où vous trouverez l’option **Transférer** le projet.
  </Accordion>
  <Accordion title="Puis-je changer le propriétaire de l’espace de travail&nbsp;?">
    Oui, un propriétaire peut ajouter plusieurs propriétaires d’espace de travail.
  </Accordion>
  <Accordion title="Puis-je changer le propriétaire du projet&nbsp;?">
    Pour le moment, ce n’est pas possible, mais vous pouvez toujours remixer ce projet vers un nouveau compte et vous serez propriétaire de cette copie.
  </Accordion>
</AccordionGroup>
