Lorsque vous envoyez des messages dans Lovable, vous pouvez choisir différents modes, adaptés à différents besoins.

<div id="agent-mode">
  ## Mode Agent
</div>

Le mode Agent permet à Lovable de réfléchir, de planifier et d’agir de manière autonome. Il est conçu pour rendre Lovable plus performant, fiable et indépendant lorsqu’il gère des tâches de développement complexes.
Le mode Agent peut :

* Interpréter votre demande
* Explorer votre base de code pour obtenir le contexte nécessaire
* Lire des fichiers à la demande
* Corriger et refactoriser le code selon les besoins
* Déboguer les problèmes de manière proactive
* Inspecter les journaux et l’activité réseau pour identifier et corriger automatiquement les erreurs
* Rechercher sur le Web en temps réel pour récupérer de la documentation, des ressources ou des captures d’écran
* Générer et modifier des images pour votre application

Le tout sans intervention manuelle. Grâce à une meilleure compréhension de votre base de code et au débogage en temps réel, il offre de meilleurs résultats avec moins d’erreurs et moins de modifications indésirables.
Ces outils permettent à Lovable d’agir avec autonomie et précision, en réduisant les frictions pendant le développement.

**Tarification**

Le mode Agent utilise une tarification à l’usage, que vous pouvez découvrir plus en détail [ici](/fr/introduction/plans-and-credits#credit-usage). Le coût d’un message dépend de sa complexité, afin que vous ne payiez que pour ce que vous utilisez réellement.
Beaucoup de messages coûtent moins d’1 crédit, tandis que les plus complexes peuvent coûter davantage.

<div id="chat-mode">
  ## Chat mode
</div>

Chat Mode est votre partenaire pour le développement et le débogage. Il vous aide à réfléchir à vos problèmes, à diagnostiquer les erreurs et à planifier votre produit directement dans Lovable.
Chaque message dans Chat mode coûte 1 crédit.

<Card title="Activer Chat mode" img="/assets/features/images/chat-mode-edit.png">
  Pour activer **Chat mode**, cliquez simplement sur « Chat » à côté de la zone où vous saisissez vos messages dans Lovable.
</Card>

Contrairement à Agent mode, Chat mode est purement conversationnel et ne peut pas modifier votre code. C&#39;est idéal pour poser des questions, planifier votre projet et déboguer.
Il fonctionne de manière agentique, ce qui signifie qu&#39;il peut raisonner en plusieurs étapes et décider quand rechercher dans les fichiers, inspecter les journaux, interroger la base de données, etc.

Si vous lui demandez de l&#39;aide pour implémenter une fonctionnalité ou pour résoudre un problème, il proposera un plan étape par étape pour atteindre le résultat souhaité.
Un bouton « Mettre le plan en œuvre » vous sera proposé. Il vous suffit de cliquer dessus et l&#39;IA passera en Agent mode et appliquera les modifications de code nécessaires.

**Principaux cas d&#39;utilisation de Chat mode**

* Débogage efficace en demandant à l&#39;IA d&#39;examiner les problèmes en profondeur avant de demander une correction.
* Planification des structures et des relations de base de données pour soutenir une architecture système robuste et efficace.
* Implémentation optimale de nouvelles fonctionnalités en identifiant les modifications minimales nécessaires.
* Demande de conseils pour améliorer et optimiser votre application.
* Compréhension et évaluation de l&#39;impact de potentielles nouvelles fonctionnalités sur la structure et le fonctionnement de l&#39;application.
