你可以前往 [Account Settings](https://lovable.dev/settings/account)，然后进入 **Labs**，在其中启用或禁用不同的 Labs 功能。

<Danger>
  这些实验性功能可能随时变更或被移除。
</Danger>

<div id="github-branch-switching">
  ## GitHub 分支切换
</div>

启用 GitHub 分支切换功能后，你可以选择 Lovable 要编辑的分支。启用后，可以按照以下步骤切换分支：

1. 进入项目的 **Settings**
2. 打开 **GitHub** 选项卡
3. 点击分支下拉菜单，选择你要切换到的分支

<Note>
  Lovable 默认与 `main` 分支进行同步
</Note>
