**Lovable API** 让你可以通过编程方式创建和分享 Lovable 应用——只需一个链接，就能把想法、提示，甚至图像转化为完整可用的应用。

它专为希望**将 Lovable 集成到工作流程中**或**在网页上的任意位置自动生成应用**的开发者、创作者和团队而设计。

<div id="what-is-the-lovable-api">
  ## 什么是 Lovable API？
</div>

Lovable API 是一套不断扩展的功能，让你能够在网页界面之外，以更多方式使用 Lovable。

首个发布的功能是 [**Build with URL**](https://docs.lovable.dev/integrations/build-with-url)，它允许你通过一个可分享的 URL 直接生成应用。

借助它，你可以：

- 在你的网站或应用中嵌入 **Build with Lovable** 按钮。
- 与团队成员或客户分享用于生成应用的链接。
- 将 Lovable 应用创建流程集成到内部工具或脚本中。
- 通过单个链接即可即时创建原型。

<div id="why-use-the-lovable-api">
  ## 为什么要使用 Lovable API？
</div>

- **自动化：** 从代码、仪表盘或工作流中触发应用的创建。
- **集成：** 将 Lovable 融入你自己的产品体验。
- **速度：** 更快地生成并分享新的应用创意。
- **易分享：** 通过 URL 轻松实现无缝协作。
