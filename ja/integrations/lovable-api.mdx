**Lovable API** を使うと、Lovable アプリをプログラムから作成・共有でき、アイデアやプロンプト、さらには画像からでも、1 つのリンクだけで完成したアプリケーションとして生成できます。

これは、**自分たちのワークフローに Lovable を統合**したり、Web 上のあらゆる場所から**アプリ生成を自動化**したい開発者、クリエイター、チーム向けに設計されています。

<div id="what-is-the-lovable-api">
  ## Lovable API とは？
</div>

Lovable API は、ウェブインターフェースにとどまらず Lovable を活用できるようにする、拡張機能群からなる API です。

最初のリリースである [**Build with URL**](https://docs.lovable.dev/integrations/build-with-url) を使うと、共有可能な URL から直接アプリを生成できます。

これにより、次のことができます：

- 自分のサイトやアプリに **Build with Lovable** ボタンを埋め込む。
- チームメイトや顧客と、アプリ生成用のリンクを共有する。
- 内部ツールやスクリプトに Lovable のアプリ作成を組み込む。
- 1 本のリンクから、すぐにプロトタイピングを行う。

<div id="why-use-the-lovable-api">
  ## なぜ Lovable API を使うのか？
</div>

- **自動化:** コードやダッシュボード、ワークフローからアプリ作成を自動実行できます。
- **連携:** Lovable を自社プロダクトの体験の中に組み込めます。
- **スピード:** これまでになく速く、新しいアプリのアイデアを生成して共有できます。
- **共有性:** URL を共有するだけで、コラボレーションをシンプルかつスムーズに行えます。
