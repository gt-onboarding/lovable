[Account Settings](https://lovable.dev/settings/account) に移動し、**Labs** を開くと、さまざまな Labs の機能を有効化または無効化できます。

<Danger>
  これらの実験的な機能は、予告なく変更されたり削除されたりする場合があります。
</Danger>

<div id="github-branch-switching">
  ## GitHub ブランチの切り替え
</div>

GitHub ブランチの切り替えを有効にすると、Lovable が編集するブランチを選択できるようになります。有効化したら、次の手順でブランチを切り替えられます。

1. プロジェクトの **Settings**（設定）に移動します
2. **GitHub** タブに移動します
3. ブランチのドロップダウンをクリックし、切り替えたいブランチを選択します

<Note>
  Lovable はデフォルトで「main」ブランチと同期します
</Note>
