Lovable は、無料プランと複数の有料プランを備えたサブスクリプション型サービスです。サブスクリプションに加入すると、利用できる機能とクレジットが増えます。Lovable でメッセージを送信するにはクレジットが必要です。

<div id="feature-comparison-free-plan-vs-paid-plans">
  ## 無料プランと有料プランの機能比較
</div>

<div id="free-plan">
  ### 無料プラン
</div>

- 1日あたり5クレジット（最大月30クレジット）
- ワークスペースでのメンバー数無制限の共同作業

<div id="pro-plan">
  ### Pro プラン
</div>

- 1日あたり5クレジット（最大月150クレジットまで）
- プランに応じた月間クレジット
- メンバー数無制限でのワークスペースでのコラボレーション
- ワークスペースのロールと権限管理
- 非公開プロジェクト
- Lovable プロジェクトにカスタムドメインを接続する機能
- Lovable プロジェクトから「Edit with Lovable」バッジを削除する機能
- Lovable 内でコードを直接編集できる Code モードへのアクセス

<div id="business-plan">
  ### ビジネスプラン
</div>

_Pro に含まれるすべての機能に加えて、次の機能が利用できます:_

- SSO
- ワークスペース内の個人用プロジェクト
- データ学習への利用のオプトアウト
- プロジェクトを効率化する再利用可能なデザインテンプレートの作成

<div id="credit-display">
  ## クレジットの表示
</div>

自分がどれくらいのクレジットを保有しているか、または使用したかを確認するには、メインダッシュボードでワークスペース名をクリックするか、プロジェクトエディタ内でプロジェクト名をクリックします。  
現在の請求期間で残っているクレジット数と、クレジットバーが表示されます。  
クレジットバーは、ユーザーに残っているクレジット数と、すでに使用されたクレジット数を視覚的に表します。 

- **グレーの部分**: 今の請求期間で、すでに使用したクレジット数を示します。 
- **青い部分**: 残りのクレジットの種類ごとの内訳を示します。各色のセクションにカーソルを重ねると、そのクレジットタイプと、そのタイプで利用可能な正確なクレジット数がツールチップで表示されます。

<div id="credit-usage">
  ## クレジットの使用
</div>

Lovable ではクレジットの使用量に基づく従量課金制を採用しており、メッセージを送信するとクレジットが消費されます。  
メッセージ 1 通あたりのコストは、その内容の複雑さに応じて決まり、実際に使った分だけ支払う仕組みになっています。

多くのメッセージは 1 クレジット未満で済みますが、より複雑なものは多くのクレジットを消費する場合があります。  
この仕組みにより、よりきめ細かな編集ができ、1 メッセージあたりの効率も高まるため、結果として Lovable をより手頃な価格で利用できます。

以下は、いくつかのプロンプト例とそのコストです。

| User Prompt                        | 実行内容                                                          | 使用クレジット     |
| ---------------------------------- | ----------------------------------------------------------------- | ------------------ |
| Make the button gray               | ボタンのスタイルを変更します                                     | 0.50               |
| Remove the footer                  | フッターコンポーネントを削除します                               | 0.90               |
| Add authentication                 | ログインと認証ロジックを追加します                               | 1.20               |
| Build a landing page with images   | 画像を自動生成し、テーマやセクションを含むランディングページを作成します | 2.00               |

<div id="credit-rollovers">
  ## クレジットの繰り越し
</div>

未使用の月次クレジットは、アクティブなサブスクリプションを継続している限り、月額プラン・年額プランのどちらでも、各請求サイクルの終了時に自動的に繰り越されます。繰り越されたクレジットは、月額プランの場合は 1 か月間利用できます。**年額プラン**のユーザーの場合、繰り越されたクレジットは、その年額プランの契約期間中ずっと利用できます。

**解約**: 有料サブスクリプションを解約すると、未使用クレジットおよび繰り越しクレジットは現在の請求期間の終了時に失効し、その後は繰り越されません。

<Warning>
  日次クレジットは繰り越されません。未使用の日次クレジットは、無料プランおよび有料プランのどちらの場合も、日をまたいで**蓄積されることはありません**。
</Warning>

サブスクリプションオプションの詳細については、[料金ページ](https://lovable.dev/pricing)をご覧ください。

<div id="faq">
  ## FAQ
</div>

<AccordionGroup>
  <Accordion title="クレジットとは何ですか？">
    クレジットは、Lovable でメッセージを送信する際に消費されます。
  </Accordion>
  <Accordion title="サブスクリプションをアップグレードするにはどうすればよいですか？">
    1. ダッシュボード右上のプロフィールアイコンをクリックし、Settings → Plans & credits を開くか、[このページ](https://lovable.dev/?settings=billing)にアクセスします。 
    2. 「credits per month」のドロップダウンから希望するクレジット数を選択し、"Upgrade" をクリックします。Stripe のポータルでアップグレードを完了します。
  </Accordion>
  <Accordion title="サブスクリプションをダウングレードするにはどうすればよいですか？">
    1. [このページ](https://lovable.dev/?settings=billing)にアクセスするか、ダッシュボード右上のプロフィールアイコンをクリックし、Settings → Plans & credits を開きます。 
    2. 「credits/month」のドロップダウンから希望するクレジット数を選択し、"Downgrade" をクリックします。 
    3. まず Lovable 上でダウングレードを完了し、その後 Stripe のポータルでも手続きを完了します。 
    現在のプランは、現在の請求期間が終了するまで有効のままです。
  </Accordion>
  <Accordion title="サブスクリプションをキャンセルするにはどうすればよいですか？">
    1. [このページ](https://lovable.dev/?settings=billing)にアクセスするか、ダッシュボード右上のプロフィールアイコンをクリックし、Settings → Plans & credits を開きます。 
    2. Manage → Downgrade to Free をクリックし、手順を完了します。 
    現在のプランは、現在の請求期間が終了するまで有効のままです。
  </Accordion>
  <Accordion title="請求情報を変更するにはどうすればよいですか？">
    請求情報を変更するには、ダッシュボード右上のプロフィールアイコンをクリックし、Settings → Plans & credits → Manage your payment preferences を開きます。Stripe のポータルにリダイレクトされ、そこで情報を変更できます。
  </Accordion>
  <Accordion title="請求書はどのようにダウンロードできますか？">
    ダッシュボード右上のプロフィールアイコンをクリックし、Settings → Plans & credits → Manage → View invoices を開くことで、過去の請求書にアクセスできます。Stripe のポータルにリダイレクトされ、そこで過去の請求書を確認できます。
  </Accordion>
  <Accordion title="クレジットを増やすにはどうすればよいですか？">
    アカウント設定でプランをアップグレードすることで、利用できるクレジット数を増やせます。
  </Accordion>
  <Accordion title="月次クレジット上限はいつリセットされますか？">
    月次クレジット上限は、請求期間がリセットされるタイミングでリセットされます。これは、サブスクリプションを開始したのと同じ日付・時刻に、毎月発生します。詳細な日時は、[Billing セクション](https://lovable.dev/?settings=billing)で確認してください。
  </Accordion>
  <Accordion title="1日のクレジット上限はいつリセットされますか？">
    1 日あたりのクレジットは、協定世界時（UTC）の毎日深夜にリセットされます。無料プランでは、1 か月あたり合計 30 クレジットまで無料で利用できます。
  </Accordion>
  <Accordion title="クレジットを追加購入できますか？">
    請求期間がリセットされる前により多くのクレジットを利用する唯一の方法は、より上位のプランにアップグレードすることです。
  </Accordion>
  <Accordion title="ワークスペースの残りクレジットはどのように確認できますか？">
    ダッシュボード右上のプロフィールアイコンをクリックするか、プロジェクトエディタ内でプロジェクト名のドロップダウンをクリックすると、残りのクレジットを確認できます。
  </Accordion>
  <Accordion title="無料クレジットを取得できますか？">
    サポートリクエスト経由での**無料クレジットの提供は行っていません**。
  </Accordion>
</AccordionGroup>

  </Accordion>
  <Accordion title="アップグレードしましたが、クレジットが付与されていません。どうすればいいですか？">
    クレジットは、クレジット上限がより高いプランにアップグレードした場合にのみ付与されます。そのようなプランにアップグレードしているにもかかわらずクレジットが付与されていない場合は、[support@lovable.dev](mailto:support@lovable.dev) までサポートへご連絡ください。
