Al enviar mensajes en Lovable puedes elegir distintos modos que son útiles para distintas tareas.

<div id="agent-mode">
  ## Modo agente
</div>

El modo agente permite que Lovable piense, planifique y actúe de forma autónoma. Está diseñado para que Lovable sea más potente, fiable e independiente al manejar tareas de desarrollo complejas.
El modo agente puede:

* Interpretar tu solicitud
* Explorar tu base de código para obtener el contexto necesario
* Leer archivos bajo demanda
* Corregir y refactorizar código según sea necesario
* Depurar problemas de forma proactiva
* Inspeccionar registros y actividad de red para identificar y corregir errores automáticamente
* Buscar en la web en tiempo real para obtener documentación, recursos o capturas de pantalla
* Generar y editar imágenes para tu aplicación

Todo sin intervención manual. Con una comprensión más profunda de la base de código y depuración en tiempo real, ofrece mejores resultados con menos errores y menos cambios no deseados.
Estas herramientas permiten que Lovable actúe con autonomía y precisión, reduciendo la fricción durante el desarrollo.

**Precios**

El modo agente utiliza un modelo de precios basado en el uso, sobre el que puedes leer más [aquí](/es/introduction/plans-and-credits#credit-usage). El costo de un mensaje depende de su complejidad, para que solo pagues por lo que realmente usas.
Muchos mensajes cuestan menos de 1 crédito, mientras que otros más complejos pueden costar más.

[Haz clic aquí para leer el anuncio!](https://lovable.dev/blog/agent)

<div id="chat-mode">
  ## Chat mode
</div>

Chat Mode es tu aliado para el desarrollo y la depuración. Te ayuda a reflexionar sobre problemas, depurar incidencias y planificar tu producto directamente en Lovable.
Cada mensaje en Chat mode consume 1 crédito.

<Card title="Activar Chat mode" img="/assets/features/images/chat-mode-edit.png">
  Para activar **Chat mode**, simplemente haz clic en &quot;Chat&quot; junto al campo donde escribes tus mensajes en Lovable.
</Card>

A diferencia de Agent mode, Chat mode es conversacional y no puede hacer cambios en tu código. Es perfecto para hacer preguntas, planificar tu proyecto y depurar.
Es agéntico, lo que significa que puede razonar a través de varios pasos y decidir cuándo buscar archivos, inspeccionar registros, hacer consultas a la base de datos y más.

Si le pides ayuda para implementar una funcionalidad o para solucionar problemas, elaborará un plan paso a paso para llegar al resultado que deseas.
Verás un botón con la etiqueta &quot;Implement the plan&quot;. Simplemente haz clic en él y la IA entrará en Agent mode e implementará los cambios de código necesarios.

**Principales casos de uso de Chat mode**

* Depuración eficiente pidiéndole a la IA que investigue los problemas en profundidad antes de solicitar una solución.
* Planificación de estructuras y relaciones de bases de datos para dar soporte a una arquitectura de sistema eficaz.
* Implementación óptima de nuevas funcionalidades identificando los cambios mínimos necesarios.
* Solicitar consejos de mejora para potenciar tu aplicación.
* Entender y evaluar cómo las posibles nuevas funcionalidades impactarán en la estructura y funcionalidad de la aplicación.
