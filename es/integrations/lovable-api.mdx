La **API de Lovable** hace posible crear y compartir apps de Lovable de forma programática, convirtiendo ideas, instrucciones o incluso imágenes en aplicaciones completamente generadas con un solo enlace.

Está diseñada para desarrolladores, creadores y equipos que quieren **integrar Lovable en sus flujos de trabajo** o **automatizar la generación de apps** desde cualquier lugar de la web.

<div id="what-is-the-lovable-api">
  ## ¿Qué es la API de Lovable?
</div>

La API de Lovable es un conjunto cada vez mayor de funcionalidades que amplían lo que puedes hacer con Lovable más allá de la interfaz web.

La primera versión, [**Build with URL**](https://docs.lovable.dev/integrations/build-with-url), te permite generar apps directamente a partir de una URL para compartir.

Con ella, puedes:

- Incrustar botones de **Build with Lovable** en tu sitio o app.
- Compartir enlaces para generar apps con tu equipo o con clientes.
- Integrar la creación de apps de Lovable en herramientas internas o scripts.
- Prototipar al instante a partir de un solo enlace.

<div id="why-use-the-lovable-api">
  ## ¿Por qué utilizar la API de Lovable?
</div>

- **Automatización:** Inicia la creación de aplicaciones desde código, paneles o flujos de trabajo.
- **Integración:** Integra Lovable en la experiencia de tu propio producto.
- **Velocidad:** Genera y comparte nuevas ideas de aplicaciones más rápido que nunca.
- **Facilidad para compartir:** Las URL hacen que colaborar sea sencillo y sin fricción.
