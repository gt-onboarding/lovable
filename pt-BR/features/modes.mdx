Ao enviar mensagens no Lovable, você pode escolher diferentes modos, cada um adequado a um tipo de tarefa.

<div id="agent-mode">
  ## Modo agente
</div>

O modo agente permite que o Lovable pense, planeje e execute ações de forma autônoma. Ele foi desenvolvido para tornar o Lovable mais capaz, confiável e independente ao lidar com tarefas de desenvolvimento complexas.
O modo agente pode:

* Interpretar sua solicitação
* Explorar sua base de código em busca do contexto necessário
* Ler arquivos sob demanda
* Corrigir e refatorar código conforme necessário
* Depurar problemas proativamente
* Inspecionar registros de log e atividade de rede para identificar e corrigir erros automaticamente
* Pesquisar na web em tempo real para buscar documentação, recursos ou capturas de tela
* Gerar e editar imagens para seu aplicativo

Tudo isso sem intervenção manual. Com um entendimento mais profundo da base de código e depuração em tempo real, ele entrega melhores resultados, com menos erros e menos alterações indesejadas.
Essas ferramentas permitem que o Lovable atue com autonomia e precisão, reduzindo o atrito durante o desenvolvimento.

**Preços**

O modo agente usa precificação baseada em uso, sobre a qual você pode saber mais [aqui](/pt-BR/introduction/plans-and-credits#credit-usage). O custo de uma mensagem depende de sua complexidade, para garantir que você pague apenas pelo que realmente usar.
Muitas mensagens custam menos de 1 crédito, enquanto as mais complexas podem custar mais.

[Clique aqui para ler o anúncio!](https://lovable.dev/blog/agent)

<div id="chat-mode">
  ## Chat mode
</div>

Chat mode é seu parceiro de desenvolvimento e depuração. Ele ajuda você a refletir sobre problemas, depurar erros e planejar seu produto diretamente no Lovable.
Cada mensagem no Chat mode consome 1 crédito.

<Card title="Ativar o Chat mode" img="/assets/features/images/chat-mode-edit.png">
  Para ativar o **Chat mode**, basta clicar em &quot;Chat&quot; ao lado de onde você insere suas mensagens no Lovable.
</Card>

Ao contrário do Agent mode, o Chat mode é conversacional e não pode fazer alterações no seu código. Ele é perfeito para fazer perguntas, planejar seu Projeto e depurar.
Ele é agentic, o que significa que consegue raciocinar em múltiplas etapas e decidir quando buscar arquivos, inspecionar registros de log, consultar o banco de dados e muito mais.

Se você pedir ajuda para implementar uma funcionalidade ou solucionar um problema, ele criará um plano passo a passo para chegar ao resultado desejado.
Você verá um botão &quot;Implement the plan&quot;. Basta pressioná-lo e a IA entrará no Agent mode e implementará as alterações de código necessárias.

**Melhores casos de uso para o Chat mode**

* Depuração eficiente, pedindo para a IA investigar problemas em profundidade antes de solicitar uma correção.
* Planejamento de estruturas e relacionamentos de banco de dados para dar suporte a uma arquitetura de sistema eficaz.
* Implementação ideal de novas funcionalidades ao identificar as alterações mínimas necessárias para a nova funcionalidade.
* Solicitar dicas de melhoria para aprimorar seu aplicativo.
* Entender e avaliar como possíveis novas funcionalidades vão impactar a estrutura e a funcionalidade do aplicativo.
