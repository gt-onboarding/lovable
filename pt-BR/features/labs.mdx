Você pode ativar ou desativar diferentes recursos do Labs acessando as suas [Configurações da conta](https://lovable.dev/settings/account) e, em seguida, indo até **Labs**.

<Danger>
  Esses recursos experimentais podem mudar ou ser removidos a qualquer momento.
</Danger>

<div id="github-branch-switching">
  ## Troca de branches no GitHub
</div>

Ao ativar a troca de branches no GitHub, você pode selecionar a branch que o Lovable vai editar. Depois de ativar, você pode trocar de branch seguindo estas etapas:

1. Vá para **Configurações** de um projeto
2. Vá para a guia **GitHub**
3. Clique no menu suspenso de branch e escolha a branch para a qual você quer mudar

<Note>
  Por padrão, o Lovable sincroniza com a branch "main"
</Note>
