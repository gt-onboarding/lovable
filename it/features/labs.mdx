Puoi abilitare e disabilitare diverse funzionalità in Labs andando alle [Impostazioni account](https://lovable.dev/settings/account) e poi selezionando **Labs**.

<Danger>
  Queste funzionalità sperimentali possono cambiare o essere rimosse in qualsiasi momento.
</Danger>

<div id="github-branch-switching">
  ## Cambio di branch GitHub
</div>

Abilitando il cambio di branch GitHub puoi selezionare il branch che Lovable modifica. Una volta abilitata questa opzione puoi cambiare branch seguendo questi passaggi.

1. Vai alle **Settings** di un Progetto
2. Vai alla scheda **GitHub**
3. Fai clic sul menu a discesa del branch e scegli il branch a cui vuoi passare

<Note>
  Per impostazione predefinita Lovable si sincronizza con il branch "main"
</Note>
